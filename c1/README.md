## 练习 1\.14
---
我觉得`for`循环是一种固定的模式，处理同类问题时比起`while`通常会更直观，并且`for`初始化的变量只在`for`的语句体中有效，不影响后续命名。

`while`相对更为灵活，此外处理同类问题时往往可以比for的代码量小一点点。
